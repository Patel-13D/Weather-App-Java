# 🌦️ Weather App (Java)

A clean and functional Android application built to provide real-time weather updates and a 5-day forecast.

## 🚀 Features
* **Real-time Data:** Fetches current weather conditions based on location.
* **5-Day Forecast:** Detailed view of upcoming weather patterns.
* **Home Screen Widget:** Includes a custom `WeatherWidgetProvider` for quick updates without opening the app.
* **Modern UI:** Built using XML layouts with a focus on user experience.

## 🛠️ Tech Stack
* **Language:** Java
* **IDE:** Android Studio
* **Version Control:** Git & GitHub (Profile: Patel-13D)
